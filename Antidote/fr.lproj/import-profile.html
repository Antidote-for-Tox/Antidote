Pour importer votre fichier Tox:

  1. Envoyer le fichier ".tox" vers votre appareil de n'importe quelle application (Mail, Dropbox, etc.).
  2. Utiliser le menu "Ouvrir Dans" pour ce fichier.
  3. Sélectionner Antidote de la liste des applications disponibles.
  4. Vérifier le nom de votre nouveau profil et appuyer sur OK.
