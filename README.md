## Usage

#### Manual Installation

Clone repo, run install script and open `Antidote.xcworkspace` file with Xcode 8.

```
git clone https://github.com/Antidote-for-Tox/Antidote.git
cd Antidote
./install.sh
open Antidote.xcworkspace
```
